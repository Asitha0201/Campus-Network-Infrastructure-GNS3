# Campus Network Infrastructure - GNS3

A Cisco GNS3-based campus network project implementing VLAN segmentation, Router-on-a-Stick, Inter-VLAN Routing, DHCP, SSH, and Port Security.

---

## Features

- VLAN Segmentation
- Router-on-a-Stick
- Inter-VLAN Routing
- DHCP
- SSH
- Port Security
- Hierarchical Network Design
